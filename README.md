Weather API CLI

A simple command-line Python application that fetches current weather data using the Open-Meteo API.

Features

* Get weather data by coordinates
* User input for latitude, longitude, and timezone
* Displays:
    * Temperature (°C)
    * Wind speed (km/h)
* Handles:
    * Network errors
    * Timeout errors
    * Invalid input
    * API response errors

Technologies Used

* Python 3
* Requests library
* Open-Meteo API

Example Input

Enter latitude: 56.50
Enter longitude: 60.35
Enter timezone: Asia/Yekaterinburg

Example Output

Temperature: 21.4 °C
Wind speed: 5.2 km/h
